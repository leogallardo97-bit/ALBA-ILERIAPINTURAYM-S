# 🏗️ KL GALLARDO — Identidad Visual y Web Premium

Repositorio oficial del proyecto de marca e infraestructura web para **KL Gallardo**, especialistas en reformas integrales, pintura técnica y albañilería de alto nivel.

## 🌟 Propuesta de Valor
Este proyecto no solo es una web, es el levantamiento de una identidad corporativa diseñada para el mercado premium. Se basa en tres pilares:
1.  **Precisión:** Reflejada en el monograma geométrico de la marca.
2.  **Transparencia:** Mediante un proceso didáctico de 4 pasos para el cliente.
3.  **Confianza:** Enfoque en la limpieza post-obra y cumplimiento riguroso de plazos.

## 🛠️ Tecnologías
- **HTML5 Semántico:** Estructura optimizada para SEO.
- **Vanilla CSS:** Diseño premium con variables personalizadas, glassmorfismo y tipografía moderna (Montserrat & Outfit).
- **Responsive Design:** Optimizado para dispositivos móviles y escritorio.
- **Integración Directa:** Botón de contacto rápido vía WhatsApp.

## 🚀 Despliegue
La página está diseñada para ser desplegada mediante **GitHub Pages**.
